UserAgentInterceptor

class UserAgentInterceptor(context: Context, module: String, version: String) : Interceptor

UserAgentInterceptor will add the necessary data to the header in against of default User-Agent before proceeding to the network to obtain the access token.

Currently, this interceptor is used by AtomSDK, BPC SDK and Core SDK

Parameters

context

Application context, used to extract application name and version

module

SDK Name to add in User-Agent, from where this interceptor is being called

version

SDK Version to add in User-Agent, from where this interceptor is being called
